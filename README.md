## Assignments:
#### Car Payment Calculator
Calculates how much a monthly payment would be depending on the principal and the amount of loan years.
Uses iomanip to set the precision of the output decimal, since money is in the form of `$~#.##`.
* carPayment.cpp

#### Data Type Example Program 
A test for different data type inputs, such as char, float, int and string.
Also includes an out-of-lesson if/else block.
* dataType.cpp

#### Grade Calculator 
Calculates the gradepoint average and the grade character from the percentage of four class quarters.
Uses if, else if, else blocks to decide the grade character.
* gradeCalc.cpp

#### Simple Math Calculator 
As the name says, a simple calculator that inclues adding, subtracting, multiplying and dividing.
* simpleMath.cpp

#### Template Test 
A test file to see if the class remembered what the template was from ./notes/fileTemp.cpp.
* tempTest.cpp

#### User Input
First name and last name user input.
* userInput.cpp
